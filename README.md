# ph
- 웹페이지 동작을 위해 우선 html 처리하여 요구사항에 맞는 json response 는 웹페이지 하단 별도 추가
- 테스트 데이터 필요시 init.sql 하단에 있는 테스트데이터 쿼리 주석 해제 후 docker image build
- `mysql` 이미지 실행에 시간 소요되어 `ph`에서 연결재시도
